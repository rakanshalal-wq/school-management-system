# school-management-system
A Python educational project for managing school classes and students using a GUI.  Includes a fully working version and a step-by-step explained version for learning purposes.


---

## Project Structure

- `school_manager.py`  
  Production-ready version of the app (GUI + data saving).

- `school_manager_explained.py`  
  Fully commented educational version, explaining the code step by step in Arabic.

- `.gitignore`  
  Standard Python ignore file.

- `LICENSE`  
  MIT license – free to use for learning and development.

---

## فكرة المشروع بالعربي

مشروع تعليمي بلغة Python لإدارة الطلاب والفصول في مدرسة باستخدام واجهة رسومية (GUI).

### مميزات المشروع:
- إضافة فصول دراسية
- إضافة طلاب لكل فصل
- حساب حالة الطالب (ناجح / راسب)
- حفظ البيانات تلقائياً بعد الإغلاق
- نسختان من المشروع:
  - نسخة تشغيلية
  - نسخة تعليمية مشروحة سطرًا بسطر
